# Интерактивный инструмент для анализа финансовых рисков

Это веб-приложение на Streamlit, которое позволяет проводить глубокий анализ рисков для финансовых активов, сравнивая три различные модели распределения доходностей: Гауссову, Леви-стабильную и GARCH-t.

## Возможности

- **Анализ любого актива:** Введите тикер с Yahoo Finance, чтобы загрузить и проанализировать данные.
- **Сравнение моделей:** Визуализируйте, как три разные модели описывают эмпирические данные о доходности (графики PDF и Q-Q).
- **Оценка Value-at-Risk (VaR):** Запустите симуляцию Монте-Карло для оценки потенциальных убытков на заданном горизонте.
- **Динамический анализ риска:** Отслеживайте изменение параметра стабильности Леви (`alpha`) во времени, чтобы увидеть, как меняется уровень экстремального риска.

## Установка

1.  **Клонируйте репозиторий или скачайте файлы.**
2.  **Создайте виртуальное окружение (рекомендуется):**
    ```bash
    python -m venv venv
    source venv/bin/activate  # Для Linux/macOS
    venv\Scripts\activate    # Для Windows
    ```
3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

## Запуск

После установки зависимостей, запустите приложение командой:

```bash
streamlit run app.py
```

Приложение откроется в вашем браузере по адресу `http://localhost:8501`.
